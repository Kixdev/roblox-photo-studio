# 🎬 Roblox Cinematic Photo Studio
Roblox Cinematic Photo Studio is a **client-side cinematic tool** designed to help creators capture **high-quality screenshots, thumbnails, and cinematic scenes** inside Roblox experiences.

This tool works **locally on your client only**.
Nothing is replicated to the server and **other players are not affected**.

It is built specifically for:

* Screenshot creators
* Thumbnail designers
* Cinematic photographers
* Roblox developers who need clean visuals

## ✨ Key Features

### 🎥 Camera Control

* Smooth **Camera FOV slider**
* FOV stays locked even if the experience tries to override it
* One-click reset to original FOV

---

### 🌗 Time Control (Day / Night)

* Change in-game time using a smooth slider
* **Time lock system** prevents the map from reverting the time
* Reset restores the current world time
* Perfect for night shots, sunset lighting, or dramatic scenes

---

### 🌫️ Depth of Field (DOF)

* Enable / disable Depth of Field instantly
* Adjust:

  * Focus Distance
  * In-Focus Radius
  * Near Blur Intensity
  * Far Blur Intensity
* Ideal for cinematic background blur and subject isolation

---

### 🎨 Color & Tone Adjustment

* Fine control over:

  * Saturation
  * Contrast
  * Exposure (Brightness)
* Allows full color grading for cinematic mood and style

---

### 👤 NameTag Control (Local Only)

* Hide your in-game name tag
* Replace display name and username with custom text
* Reset instantly restores original name tags
* Only visible **to you**, safe for public servers

---

### 🖱️ Professional Cinematic UI

* Clean **ImGui-style interface**
* Header is fully draggable
* Smooth slider behavior:

  * Click, hold, and drag (no snapping or slipping)
* Sliders respond smoothly on **PC and mobile**
* Values update in real time

---

### ⌨️ Quick Capture Mode

* Press **`,` (comma)** to instantly hide or show the entire UI
* Perfect for clean screenshots without closing the tool

---

## 📱 Mobile Support

* Fully responsive UI
* Sliders work correctly with **touch input**
* Dragging and scrolling are optimized for Android devices

---

## 🔒 Safety & Compatibility

* Client-side only (LocalScript behavior)
* No server interaction
* No replication
* Safe to use in public experiences
* Does not modify game assets or data

## 🚀 Installation

1. Execute the script using your preferred executor
2. The UI will appear in-game
3. Adjust camera, lighting, and effects
4. Press **`,`** to hide UI
5. Capture your shot
